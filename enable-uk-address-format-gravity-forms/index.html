Enable UK address format in Gravity Forms

The below is a collection of functions to alter address formatting in Gravity Forms. The built-in format isn’t very helpful if you’re in the UK, so this changes the labels and allows a “UK/Ireland” choice to be made in the address field of your form.

Include $formid if you just want to target one form.
